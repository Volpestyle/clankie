# ADR 0010: Embodied teammate; persona is data, never authority

Status: accepted.

The product is a fleet OS embodied as one persistent character with memory and presence in the team's social space (Discord text/voice first). Presence is ambient supervision—the attention queue projected into the room—and personality is the trust layer that keeps supervision cheap. Persona definitions (`soul.md`) and skin packs are model-controlled data that shape tone and presentation only; they can never alter doctrine, permissions, routing, or evidence requirements. Ambient channels carry reduced command authority: steer/query/pause only, with approvals restricted to authenticated surfaces.
