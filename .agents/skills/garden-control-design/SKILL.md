# Garden control design

- Map every visual object to a canonical mission/task/worker/artifact ID.
- Keep project biomes spatially stable; use stations for dynamic workflow phase.
- Encode harness with color plus glyph, role with accessory, state with animation, attention with outline, and authority/risk with a separate signal.
- Animation follows operational state and never causes it.
- Every gesture compiles to a typed command and policy preview.
- Provide an accessible list/inspector equivalent for every canvas action.
- Optimize fleet operations: multi-select, control groups, attention queue, batch commands, synchronized graph/terminal focus.
- Never add simulated friction to live operations.
- Reward verified outcomes, not tokens, lines, agent count, or apparent busyness.
